## Features

- Firebase Authentication
- User Registration and Login
- Email Verification
- Profile Management
- Bottom Navigation

## Tech Stack

- Kotlin
- Firebase Authentication
- Jetpack Compose
- Hilt (Dependency Injection)

## Getting Started

1. Clone the repository.
2. Open in Android Studio.
3. Run on an emulator or physical device.
